Instructions

1. Cmd + Shift + click here to open the Facebook Tinder auth workflow

2. Cmd + Shift + C in that tab to open the developer console

3. If you don't see a "Continue" button, select the icon that looks like an iPhone in the upper left corner of the developer console ("Toggle device toolbar") and refresh the page.

4. Click "Continue"

5. At the top of the developer console, select the "network" tab, then select the "confirm/" entry in the table under "Name". Finally, select the "response" tab and copy the entire text string that appears underneath.

6. Paste the text below and press Submit.
